# 카친캠 기능 명세

카친캠(카카오 친구 캠페인)은 카카오톡 친구 대상 마케팅 메시지를 발송하는 시스템입니다.

---

## 2. 수신자 필터 (cafe24 스타일 테이블 폼)

| 필터명 | 입력 방식 |
|---|---|
| 회원등급 | 드롭다운 (전체/VIP/골드/실버/일반/신규) |
| 가입일 | 범위(X~Y일) / 전체 라디오 |
| 마지막 로그인(일) | 범위(X~Y일이 된) / 전체 라디오 |
| 구매여부 | 라디오 (구매이력있음/없음/전체) |
| 구매이력 | 마지막 구매 이후 X~Y일 / 전체 |
| 장바구니 상품 보유 | 라디오 (보유/미보유/전체) |
| 장바구니 담은기간(일) | 범위 (X~Y일) |
| 휴면회원 전환 기간 | 날짜 범위 |

## 3. 메시지 유형 (2가지)

| 유형 | 설명 | 단가 |
|---|---|---|
| 기본형 | 텍스트만 또는 텍스트+이미지(선택). 이미지 추가 시 이미지형으로 발송 | 25P/건 |
| 와이드 이미지형 | 전체 폭 이미지 (800×600px, 본문 76자, 버튼 2개) | 25P/건 |

## 4. 캠페인 등록 4단계

1. **캠페인 유형** — 12종 카드 중 택 1 + 서브타입 선택 (해당 시)
2. **메시지 작성** — 유형 선택 + 본문 + 이미지(선택) + 버튼(링크형/쿠폰형) + 개인화 변수 + AI 문구 추천
3. **오디언스 설정** — 유형별 필터 조건 입력 (cafe24 테이블 폼)
4. **발송 조건 설정** — 횟수(1회/반복) + 발송시각 + 야간발송 제한 안내

## 5. 서비스 연동 필수

- 카카오 브랜드메시지 서비스 연동 후 캠페인 만들기 가능
- 미연동 시 안내 화면 + "서비스 연동하기" 버튼
- 연동 상태: useServiceStore (persist)
