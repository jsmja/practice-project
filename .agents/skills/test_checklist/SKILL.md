# 테스트 체크리스트

## 1. 배너 관리

- [ ] 배너 만들기 5단계 위자드 진행 (유형→템플릿→콘텐츠→오디언스→발송조건)
- [ ] 배너 유형 5종 선택 + 세부유형 선택 (해당 시)
- [ ] 템플릿 선택 시 우측 추천 문구 표시 + 선택 시 미리보기 변경
- [ ] 콘텐츠 설정 세부유형별 UI (기본형/프로모션형/유튜브/인스타/수신동의)
- [ ] 오디언스 설정: 타겟팅 추가/삭제/초기화, 추천 가이드 표시
- [ ] 발송 조건 설정: 횟수/우선순위/노출위치 + 우측 위치 미리보기
- [ ] 스텝 자유 이동 (방문한 스텝 간)
- [ ] 3단 리사이즈 (좌/중앙/우)
- [ ] 진행 상태 바 모든 스텝에서 표시

## 2. 메시지 관리 (캠페인)

- [ ] 서비스 미연동 시 안내 화면 + 서비스 연동 버튼
- [ ] 개발자 모드 토글 (연동/미연동 전환)
- [ ] 캠페인 만들기 4단계 (유형→메시지작성→오디언스→발송조건)
- [ ] 12종 캠페인 유형 + 서브타입 선택
- [ ] 메시지 작성: 기본형/와이드 이미지형 + 가격 안내
- [ ] AI 문구 추천, 개인화 변수 삽입, A/B 테스트
- [ ] 오디언스: cafe24 테이블 폼 + 캠페인별 추천 가이드
- [ ] 발송 조건: 횟수(1회/반복) + 발송시각 + 야간 제한 안내
- [ ] 우측 미리보기: 카카오톡 UI + 포인트 정보 + 진행 상태
- [ ] 실패 상태 툴팁으로 실패 사유 확인
- [ ] 포인트 잔액 + 충전하기 버튼 표시

## 3. 서비스 연동

- [ ] 카카오 브랜드메시지 신청 폼 (채널ID/산업분류/비즈인증/인증번호)
- [ ] 비즈니스 채널 인증 체크박스 (미체크 시 경고)
- [ ] 툴팁: 모바일 메시지 수신동의 항목, 080 수신거부
- [ ] 약관 동의 (개인정보 수집, 이용약관 상세 내용)
- [ ] 신청 완료 → 연동 상태 전환 (useServiceStore)
- [ ] 연동 완료 시 상세 페이지 (연동 정보/발송 현황/해지/캠페인 이동)
- [ ] 연동 해지 → 미연동 상태로 전환

## 4. 포인트 관리

- [ ] 포인트 현황: 사용 가능 잔액 / 환불 대기 분리 표시
- [ ] 충전: 금액 선택 + 수수료(10%) + 카드 선택 (기존/신규)
- [ ] 환불: 카드 즉시 환불 / 계좌 환불 (운영팀 승인)
- [ ] 환불 대기 중 복수 환불 불가
- [ ] 환불 현황 접기 UI + 신청 취소
- [ ] 잔액 경고 배너 (50,000P 이하 / 10,000P 이하)

## 5. 결제내역

- [ ] 검색 기간 (시작일~종료일) + 전체/주/월 퀵필터
- [ ] 합계 카드: 결제/충전 합계, 환불/취소 합계, 순 결제 금액
- [ ] 상태 탭: 결제완료/충전완료/환불대기/환불완료/결제취소/결제실패
- [ ] 환불대기 건 "환불 처리" 버튼
- [ ] 환불 금액 빨간색 표시

## 6. 구독 서비스

- [ ] 서비스 신청 2단계: 서비스 선택 → 카드 선택/결제
- [ ] 기존 카드 / 신규 카드 선택
- [ ] 구독현황: 결제 수단 정보

## 7. 회사 정보 관리

- [ ] 회사명/대표자명/홈페이지URL/사업자명/업태업종 입력
- [ ] 수정/저장/취소 모드 전환
- [ ] 업태업종 변경 시 배너 추천 템플릿 연동

## 8. 공통 체크

- [ ] 모든 페이지 PageHeader 아래 space-y-6 간격 통일
- [ ] 보라색 컬러 시스템 일관성 (bg-primary, border-primary)
- [ ] GNB 메뉴 순서: 대시보드 / 헤이보드·마케팅 / 구독서비스·결제포인트 / 통계 / 고객·멤버·설정
