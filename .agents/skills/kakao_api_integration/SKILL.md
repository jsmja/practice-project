# 📱 카카오 API 연동

이 스킬은 카카오톡 비즈메시지 발송과 관련된 외부 API 연동 규칙을 정의합니다.
외부 시스템 연동이므로 별도의 규칙이 필요합니다.

---

## 1. 카카오 비즈메시지 API 개요

### 1.1 인증
| 항목 | 설명 |
|---|---|
| 인증 방식 | API Key + Access Token |
| 토큰 갱신 | 만료 전 자동 갱신 로직 필수 |
| 환경 분리 | 개발/QA/운영 환경별 API Key 분리 |

### 1.2 주요 엔드포인트

| 기능 | Method | Path | 설명 |
|---|---|---|---|
| 토큰 발급 | POST | `/oauth/token` | Access Token 발급 |
| 메시지 발송 | POST | `/v2/api/talk/friends/message/send` | 친구 대상 메시지 발송 |
| 발송 결과 조회 | GET | `/v2/api/talk/friends/message/result` | 발송 결과 확인 |
| 친구 목록 조회 | GET | `/v1/api/talk/friends` | 전체 친구 수/목록 조회 |

## 2. 메시지 타입별 제약

### 2.1 텍스트 메시지
| 항목 | 제약 |
|---|---|
| 최대 글자 수 | **400자** |
| 버튼 | 최대 5개 |
| 이미지 | 없음 |

### 2.2 이미지 메시지
| 항목 | 제약 |
|---|---|
| 최대 글자 수 | **400자** (이미지 아래 텍스트) |
| 이미지 크기 | 최대 500KB |
| 이미지 포맷 | JPG, PNG |
| 이미지 비율 | 2:1 권장 |
| 버튼 | 최대 5개 |

### 2.3 와이드 메시지
| 항목 | 제약 |
|---|---|
| 최대 글자 수 | **76자** |
| 이미지 크기 | 최대 500KB |
| 이미지 비율 | 카카오 와이드형 규격 |
| 버튼 | 최대 2개 |

> [!WARNING]
> 글자 수 초과 시 카카오 API에서 **발송 거부**됩니다.
> 반드시 프론트엔드에서 사전 검증 후 API를 호출해야 합니다.

## 3. 에러 코드 처리

### 3.1 에러 코드 분류

| 에러 코드 | 분류 | 설명 | 처리 방법 |
|---|---|---|---|
| `1001` | 인증 오류 | 토큰 만료 | **자동 재시도** (토큰 갱신 후) |
| `2001` | 발송 오류 | 수신 거부 사용자 | **포인트 환불** |
| `2002` | 발송 오류 | 차단된 사용자 | **포인트 환불** |
| `2003` | 발송 오류 | 존재하지 않는 사용자 | **포인트 환불** |
| `3001` | 제한 초과 | 일일 발송 한도 초과 | **발송 중지 + 사용자 알림** |
| `3002` | 제한 초과 | 분당 발송 한도 초과 | **자동 재시도** (대기 후) |
| `4001` | 메시지 오류 | 글자 수 초과 | **발송 불가** (사전 검증 필요) |
| `4002` | 메시지 오류 | 이미지 규격 불일치 | **발송 불가** (사전 검증 필요) |
| `5001` | 서버 오류 | 카카오 서버 장애 | **자동 재시도** (최대 3회) |

### 3.2 자동 재시도 vs 포인트 환불 분기

```
에러 발생
  ├── 재시도 가능한 에러 (1001, 3002, 5001)
  │   └── 최대 3회 재시도
  │       ├── 성공 → 정상 처리
  │       └── 3회 실패 → 포인트 환불
  │
  └── 재시도 불가능한 에러 (2001, 2002, 2003, 3001, 4001, 4002)
      └── 즉시 포인트 환불 + 결과 기록
```

### 3.3 재시도 규칙
| 항목 | 값 |
|---|---|
| 최대 재시도 횟수 | 3회 |
| 재시도 간격 | 지수 백오프 (1초, 2초, 4초) |
| 타임아웃 | 요청당 30초 |

## 4. 친구 목록 조회

### 4.1 전체 친구 수 조회
- 캠페인 발송 전 대상 수 확인 용도
- 필터 조건 적용 후의 예상 수신자 수 계산에 사용

### 4.2 페이징 처리
| 항목 | 값 |
|---|---|
| 1회 조회 최대 | 100명 |
| 페이징 방식 | 커서 기반 |
| 전체 조회 | 커서 반복 호출 |

### 4.3 친구 정보 필드
| 필드 | 설명 |
|---|---|
| `userId` | 카카오 사용자 ID |
| `profileNickname` | 프로필 닉네임 |
| `profileImageUrl` | 프로필 이미지 URL |
| `friendAddedAt` | 친구 추가 일시 |
