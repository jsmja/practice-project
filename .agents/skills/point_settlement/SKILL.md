# 포인트 정산 로직

## 1. 기본 정보

- 1 포인트 = 1원
- 유효기간: 충전일로부터 1년
- 발송 단가: **25P/건** (기본형·와이드 이미지형 동일)
- 최소 충전: 10,000P (10,000원 단위)

## 2. 충전 프로세스

- 충전 금액 선택: 1만 / 3만 / 5만 / 10만 / 30만원 + 직접 입력
- **수수료: 10%** (충전 금액 × 0.1)
- 총 결제 금액 = 충전 금액 + 수수료
- 결제 수단: 기존 등록 카드 / 신규 카드 등록 선택
- 충전 즉시 반영

## 3. 차감 로직

- 메시지 발송 시 (대상 수 × 25P) 차감
- 잔액 부족 시 발송 중단
- 발송 실패 건수만큼 자동 환불

## 4. 환불 프로세스

### 카드 즉시 환불
- 결제 금액 전액 → 결제 카드로 즉시 환불

### 계좌 환불 (운영팀 승인)
- 부분 환불 가능, 계좌 정보 입력 필수
- 상태 흐름: 환불신청 → 환불대기 → 환불완료
- 처리 기간: 영업일 3~5일

### 환불 제한
- 복수 환불 불가 (처리 중 건이 있으면 신규 불가)
- 환불 대기 금액 = 출금 보류 (사용 가능 잔액에서 차감)
- 환불 신청 취소 가능

## 5. 잔액 관리

- 사용 가능 잔액 = 총 잔액 - 환불 대기
- 잔액 ≤ 50,000P: 노란 경고
- 잔액 ≤ 10,000P: 빨간 경고

## 6. 결제내역 통합

결제내역에 구독 결제 + 포인트 충전/환불 모두 표시.
검색 기간 필터 + 합계 카드 + 환불대기 건 수동 처리 버튼.
