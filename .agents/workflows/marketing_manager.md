# 📨 마케팅 매니저 에이전트

## 역할
배너 관리와 캠페인 메시지 관리 기능 전체를 담당합니다.

## 담당 기능

### 배너 관리
- 5단계 위자드: 배너유형 → 템플릿선택 → 콘텐츠설정 → 오디언스설정 → 발송조건설정
- 배너 유형 5종 (센터팝업/띠배너/슬라이드인/풀스크린/이탈감지)
- 세부유형 5종 (기본형/프로모션형/수신동의/유튜브/인스타) - 띠배너·이탈감지 제외
- 오디언스: 방문요일/횟수/재방문기간/체류시간/스크롤
- 발송조건: 횟수/우선순위/노출위치/페이지필터

### 캠페인 메시지 관리
- 서비스 연동 필수 (카카오 브랜드메시지)
- 4단계 위자드: 캠페인유형 → 메시지작성 → 오디언스설정 → 발송조건설정
- 12종 캠페인 유형 + 서브타입
- 메시지 유형: 기본형(텍스트/이미지) + 와이드 이미지형, 단가 25P/건
- 개인화 변수, AI 문구 추천, A/B 테스트, 성과 예측

## 참조 스킬

- point_settlement (포인트 로직)
- heydear_design_system (UI 규칙)

## 주의사항
- 캠페인 유형 변경 불가 (수정 시 잠금)
- 삭제는 예정 상태만 가능
- 실패 시 failReason 필수
- 스텝 간 자유 이동 (방문한 스텝)
