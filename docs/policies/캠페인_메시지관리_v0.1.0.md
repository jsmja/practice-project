### ▼ A-1. 캠페인 현황 (리스트)

- 설명: 생성된 캠페인을 관리하고 발송 현황을 확인

#### 화면 구성

- 포인트 잔액 카드
  - 예약 건 없을 때: `포인트 잔액 48,500P`
  - 예약 건 있을 때: `사용 가능 28,000P | 예약 차감 예정 -20,500P | 총 잔액 48,500P`
  - 충전하기 버튼 → 포인트 관리 페이지 이동
- StatCard 5종: 진행 / 예정 / 종료 / 중지 / 실패
- 유형 필터 (드롭다운)
- 캠페인 리스트 테이블

#### 리스트 항목

| 항목 | 설명 |
|------|------|
| NO | 번호 |
| 캠페인명 | 이름 |
| 캠페인 유형 | 12종 뱃지 |
| 발송 회차 | 1회 또는 반복(N/M회차) |
| 발송 일시 | 발송 시간 |
| 상태 | 진행/예정/종료/중지/실패 |
| 발송 | 성공수/대상수 (예: 3,136/3,200) |
| 발송률 | 퍼센트 (색상: 90%↑초록, 50~90%기본, 50%↓빨강) |

#### 상태별 정의

| 상태 | 설명 | 비고 |
|------|------|------|
| 진행 | 발송 진행 중 | - |
| 예정 | 발송 예정 | 예약 차감 포인트 계산 포함 |
| 종료 | 발송 완료 | - |
| 중지 | 일시 중지 | - |
| 실패 | 발송 실패 | ⚠ 아이콘 + 호버 시 failReason 툴팁 |

### ▼ A-3. 캠페인 만들기 - 2단계: 메시지 작성

- 설명: 메시지 유형 선택 및 내용 작성

#### 안내 문구 (상단)

- (광고) 자동 표시, 수신거부 안내 자동 삽입
- 수정은 발송 30분 전까지만 가능
- 20세 이상 성인 전용 메시지 체크 안내

#### 메시지 유형 (2종, 2열 그리드)

| 유형 | 설명 | 단가 |
|------|------|------|
| 기본형 | 텍스트 + 이미지(선택). 이미지 없으면 텍스트형, 있으면 이미지형 | 25P/건 |
| 와이드 이미지형 | 800×600px 전체 폭 이미지, 본문 76자, 버튼 2개 | 25P/건 |

#### 본문 편집기

- 텍스트 입력 (최대 400자 / 와이드 76자)
- 글자수 카운터: `[ 71 / 400 ]`

#### 도구 모음

| 항목 | 설명 |
|------|------|
| 버튼추가 * | 링크형 / 쿠폰형 탭 |
| 개인화 변수 | #{고객명}, #{브랜드명}, #{최근구매상품}, #{포인트잔액}, #{쿠폰코드}, #{회원등급} |
| 이모티콘 | 카카오 이모티콘 명령어 / 시스템 이모지 |
| 성인 전용 메시지 | 체크박스 |

#### AI 문구 추천

- "문구 생성하기" 버튼 → 4종 추천 문구
- 클릭 시 본문에 자동 삽입

#### A/B 테스트

- 토글 활성화 시 변형 B 메시지 입력 영역
- 50:50 분할 발송

#### 발송 성과 예측

- 예상 도달률 / 오픈률 / 클릭률

### ▼ A-4. 캠페인 만들기 - 3단계: 오디언스 설정

- 설명: 캠페인 유형별 수신 대상 필터 설정

#### 특수 케이스

- **전체 친구 대상**: 필터 없음, "카카오 채널 내 모든 친구에게 발송됩니다" 안내
- **생일 축하**: 자동 대상, "생일 고객 자동 대상" 표시

#### 필터 항목 (cafe24 테이블 폼)

| 필터 | 입력 방식 | 적용 유형 |
|------|---------|---------|
| 회원등급 | 드롭다운 | 커스텀, 웰컴백, 신규, 장기미접속, 장바구니, 장기미구매, VIP, 시즌 |
| 가입일 | 범위(X~Y일)/전체 라디오 | 신규, 장기미접속, 장바구니, 장기미구매, 커스텀, 시즌 |
| 마지막 로그인(일) | 범위(X~Y일)/전체 라디오 | 신규, 장기미접속, 장바구니, 장기미구매, 커스텀, 시즌 |
| 구매여부 | 라디오(있음/없음/전체) | 신규, 장기미구매, 커스텀, 시즌 |
| 구매이력 | 마지막 구매 이후 X~Y일/전체 | 장기미구매 |
| 장바구니 상품 보유 | 라디오(보유/미보유/전체) | 장바구니, 장기미구매, 커스텀 |
| 장바구니 담은기간(일) | 범위(X~Y일) | 장바구니 |
| 휴면회원 전환 기간 | 날짜 범위 | 웰컴백 |

#### 추천 오디언스 가이드 (캠페인 유형별)

- 하단에 캠페인 유형에 맞는 추천 설정 팁 표시
- 예: 신규회원 → "가입 후 1~27일 이내 고객을 타겟팅하면 첫 구매 전환율이 높아져요"

## B. 서비스 연동 - 카카오 브랜드메시지

### ▼ B-1. 서비스 신청

- 메뉴: 서비스 연동 > 카카오 브랜드메시지

#### 신청 정보

| 항목 | 설명 |
|------|------|
| 카카오채널 검색용 아이디 | @아이디 입력 |
| 채널 산업 분류 | 9종 드롭다운 (음식/쇼핑/생활교육/건강병원/여행숙박/문화여가/부동산금융/방송언론/서비스기타) |
| 비즈니스 채널 인증 확인 | 체크박스 (미체크 시 경고 + 신청 불가) |
| 휴대폰번호 인증 | 번호 입력 + 인증번호 요청/확인 |
| 모바일 메시지 수신동의 항목 | 파일 업로드 (jpg/png, 5MB) + 툴팁 |
| 080 무료수신거부 | 번호 입력 + 툴팁 |

#### 약관 동의

1. 개인정보 수집 이용 동의서 (필수) — 수집 목적/항목/보유기간
2. 카카오 메시지 이용약관 (필수) — 5개 조항 (신원정보/수신동의/책임/금지/템플릿)

### ▼ B-2. 연동 완료 상세 페이지

- 연동 완료 시 진입하는 상세 화면

#### 화면 구성

- 연동 정보: 채널 아이디, 산업 분류, 인증 상태, 연동일, 080 번호
- 발송 현황: 총 발송 건수, 성공률, 포인트 잔액
- 연동 해지하기 버튼 → 미연동 상태 전환
- 캠페인 관리로 이동 버튼
