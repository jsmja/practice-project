### ▼ A-0. 공통

- 메뉴 위치: 마케팅 > 배너 관리
- 배너 만들기/수정 시 3컬럼 위자드 에디터 사용
  - 좌측: 스텝 네비게이션 + 타겟팅 요약 + 이전/다음 버튼
  - 가운데: 에디터 콘텐츠
  - 우측: 미리보기 패널
- 3컬럼 가로 리사이즈 지원 (좌 56~600px / 우 56~800px / 가운데 최소 280px)
- 스텝 간 자유 이동 (방문한 스텝 간)
- 좌측 축소 시 아이콘만 표시 (120px 이하)

### ▼ A-1. 배너 현황 (리스트)

- 설명: 생성된 배너를 관리하고 상태를 확인하는 화면

#### 화면 구성

- 상단: PageHeader (마케팅 배너 관리 + "배너 만들기" 버튼)
- StatCard 4종: 운영중 / 일시중지 / 임시저장 / 종료
  - 클릭 시 해당 상태 필터링 (토글)
- 배너 리스트 테이블
  - 항목: NO, 배너명, 업종, 목적, 배너유형, 대상고객, 상태, 노출수, 클릭수, CTR, 시작일, 종료일

#### 상태별 정의

| 상태 | 설명 |
|------|------|
| 운영중 | 현재 노출 중인 배너 |
| 일시중지 | 일시적으로 노출 중단 |
| 임시저장 | 작성 중 임시 저장 |
| 종료 | 노출 기간 종료 |

---

### ▼ A-2. 배너 만들기 - 1단계: 배너 유형

- 설명: 배너 노출 방식과 세부 유형을 선택

#### 배너 유형 (5종)

| 유형 | 설명 | 세부유형 필요 |
|------|------|---------|
| 센터 팝업 | 화면 중앙 모달 팝업 | O |
| 띠배너 | 페이지 상단 얇은 띠 | X |
| 슬라이드인 | 우측 하단 슬라이드 | O |
| 풀스크린 | 전체 화면 오버레이 | O |
| 이탈 감지 | 이탈 시 자동 노출 | X |

#### 세부유형 (5종) — 띠배너/이탈감지 제외

| 세부유형 | 설명 |
|---------|------|
| 기본형 | 인사말, 브랜드 로고 |
| 프로모션형 | 이벤트, 버튼 CTA |
| 마케팅 수신동의 | 수신 동의 유도 |
| 홍보 유튜브 | YouTube 영상 노출 |
| 홍보 인스타 | Instagram 피드 노출 |

#### 우측 미리보기

- 배너 유형 선택 시 애니메이션 프리뷰 (미니 브라우저)
- 진행 상태 바 표시

---

### ▼ A-3. 배너 만들기 - 2단계: 템플릿 선택

- 설명: 업종/목적에 맞는 템플릿을 선택하고 추천 문구 확인

#### 화면 구성

- 목적 필터 (전체/신규고객유입/재방문유도/이벤트안내/신상품홍보/할인프로모션/리뷰유도/회원가입유도)
- 추천 템플릿 섹션: 회사 업태/업종 매칭 시 상단 분리 표시 (amber 보더)
- 템플릿 카드: 카카오톡 채팅 프리뷰 스타일 (배너이미지+제목+본문+버튼+푸터)

#### 우측 미리보기

- 템플릿 선택 시 추천 문구 4종 표시
- 문구 선택 시 상단 미리보기 이미지 색상 변경
- 예시 프롬프트 자동 생성

---

### ▼ A-4. 배너 만들기 - 3단계: 콘텐츠 설정

- 설명: 세부유형별 콘텐츠 상세 입력

#### 기본형

| 항목 | 설명 |
|------|------|
| 아이콘 이미지 등록 | 46×46px, 1:1 비율, JPG/PNG/GIF, 5MB |
| 아이콘 배경 컬러 | 8색 선택 (기본 #717171) |
| 메인 타이틀 * | 최대 20자 |
| 서브 타이틀 | 최대 21자 |
| 문구 입력 | 리치 텍스트 에디터, 최대 200자 |

#### 프로모션형

| 항목 | 설명 |
|------|------|
| 카드 탭 | 카드 추가/전환 가능 |
| 이미지 등록 * | 360×240px, 3:2 비율, 5MB |
| 메인 타이틀 * | 최대 48자 |
| 문구 입력 | 리치 텍스트, 최대 200자 |
| 버튼명 | 최대 20자 |

#### 홍보 유튜브

- 유튜브 영상 URL 입력
- 자동 재생 토글

#### 홍보 인스타

- 인스타그램 피드 유형: 최신순 / 특정 게시물 선택
- 최신순: 최대 24개 표시

#### 마케팅 수신동의

- 이미지 등록 (360×240px)
- 메인 타이틀 (최대 20자)
- 문구 입력 (최대 200자)
- 버튼: 링크형/쿠폰형 탭
  - 쿠폰형: 쿠폰 선택 + 발급 조건 + 쿠폰 상세
  - 링크형: 버튼 URL
- 버튼명 (최대 25자)
- 버튼 컬러

#### 우측 미리보기

- 세부유형별 실시간 콘텐츠 미리보기
- 진행 상태 바

---

### ▼ A-5. 배너 만들기 - 4단계: 오디언스 설정

- 설명: 배너 노출 대상 오디언스를 설정

#### 대상 고객

- 전체 / 로그인 / 비로그인 라디오 선택

#### 타겟팅 항목 (5종)

| 항목 | 설명 | 기본 포함 |
|------|------|---------|
| 방문 요일 | 요일별 선택 + 모든 요일 | O |
| 방문 횟수 | 숫자 입력 + 프리셋 (1/3/5/10회 이상) | O |
| 재방문 기간 | 숫자 입력 + 프리셋 (24시간/1일/3일/5일/10일 이후) | X |
| 체류 시간 | 체류 시간/특정 시간대 + 프리셋 (1/3/5/10분) | X |
| 마우스 스크롤 | 퍼센트 입력 + 프리셋 (10/30/50/80%) | X |

- 타겟팅 추가: 드롭다운 메뉴로 미사용 항목 추가
- 초기화: 전체 초기화
- 각 항목 개별 삭제 가능

#### 추천 오디언스 가이드

- 재방문 유도: 방문 요일 평일, 방문 횟수 3회 이상
- 신규 고객 확보: 방문 횟수 1회, 재방문 기간 24시간 이내
- 이탈 방지: 재방문 기간 5일 이후, 마우스 스크롤 80% 이상

#### 좌측 타겟팅 요약

- 접기/펼치기 형태
- 대상, 요일, 횟수, 재방문, 체류, 스크롤 요약 표시

#### 우측 미리보기

- 콘텐츠 미리보기 유지 + 오디언스 설정 요약 카드

---

### ▼ A-6. 배너 만들기 - 5단계: 발송 조건 설정

- 설명: 배너 노출 스케줄과 조건을 설정

#### 화면 구성

| 항목 | 설명 |
|------|------|
| 발송 시작 및 종료 설정 | 날짜 선택 드롭다운 (즉시 시작 / 날짜 지정) |
| 마케팅 발송 횟수 | 처음 1회 / 매일 1회 / 접속 할 때마다 (세그먼트) |
| 우선 순위 설정 | 1 / 2 / 3 (세그먼트) |
| 노출 위치 설정 | 페이지 가운데 / 딥처리 및 페이지 가운데 / 헤이버튼 위치 (아이콘 카드 3종) |
| 특정 페이지 방문 | URL 입력 + 적용 버튼 |
| 특정 페이지 제외 | URL 입력 + 적용 버튼 |

#### 우측 미리보기

- 미니 브라우저 안에서 노출 위치별 배너 표시
  - 센터: 딤 + 중앙 모달
  - 딥처리 및 가운데: 딤 + 중앙 모달 (약한)
  - 헤이버튼: 우하단 원형 버튼
- 발송 조건 요약 (횟수/우선순위/노출위치)
- 진행 상태 바
