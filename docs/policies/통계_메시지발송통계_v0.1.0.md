### ▼ A-1. 발송 및 클릭 탭

#### KPI 카드 (3종)

| KPI | 설명 | 색상 |
|------|------|------|
| 발송률 | 발송 성공 / 시도 | 파랑 배경 |
| 열람률 | 열람 / 발송 | 보라 배경 |
| 클릭률 | 클릭 / 열람 | 초록 배경 |

- 각 KPI에 업종 평균 범위 표시
- 평가 뱃지: 우수(초록) / 양호(파랑) / 개선 필요(주황)

#### 상세 지표 (4행)

| 행 | 항목 |
|------|------|
| 1행 | 총 시도 → 총 발송 (발송률 %) |
| 2행 | 총 발송 → 총 열람 (열람률 %) |
| 3행 | 총 열람 → 총 클릭 (클릭률 %) |
| 4행 | 클릭률 / 전환률 표시 |

#### 단계별 분석

| STEP | 항목 | 아이콘 | 비고 |
|------|------|-------|------|
| 1 | 열람률 | 👁️ | - |
| 2 | 클릭률 | 👆 | 자체측정 |
| 3 | 전환률 | 🛒 | 자체측정 |

- 각 단계: 현재 값, 프로그레스 바, 전환 건수, 이탈 건수, 업종 평균

#### 단계별 전환 흐름 시각화 (FunnelLineChart)

- SVG 차트 (900×180, 표시 240px)
- **내 실적 라인** (보라 solid): 발송 → 열람 → 클릭 → 전환
- **업종 평균 라인** (주황 점선): 평균 중간값 연결
- **평균 범위 영역** (연주황 배경): min~max 범위
- 각 포인트:
  - 실적 값: `95.2%` (보라 볼드)
  - 평균 대비: `▲ 2.5%p` (초록) 또는 `▼ 1.3%p` (빨강)
  - 평균값: `평균 14%` (주황)
  - 건수: `3,200건`
  - 평균 범위: `(평균 8-20%)`
- 하단 범례: 내 실적(보라) / 업종 평균(주황 점선) / 평균 범위(연주황)

#### 업종 평균 범위 기준

| 단계 | 평균 범위 |
|------|---------|
| 발송 | 90-99% |
| 열람 | 8-20% |
| 클릭 | 3-18% |
| 전환 | 1-8% |

#### 핵심 인사이트

- 노란 배경 박스에 KPI 분석 요약
- 예: "열람률이 업종 평균보다 높습니다. 클릭률 개선에 집중하세요."

#### 캠페인 유형별 통계

- 유형별 카드 (최대 6종)
- 각 카드: 유형명, 발송 건수, 성공률, 프로그레스 바

#### 캠페인 상세 데이터 테이블

| 항목 | 설명 |
|------|------|
| NO | 번호 |
| 캠페인명 | 이름 |
| 유형 | 뱃지 |
| 회차 | 발송 회차 |
| 발송일 | 일시 |
| 대상 수 | 타겟 인원 |
| 성공 수 | 발송 성공 |
| 실패 수 | 발송 실패 |
| 성공률 | 퍼센트 |
| 비용(P) | 차감 포인트 |

### ▼ A-2. 쿠폰 발급 및 사용 탭

#### 쿠폰 KPI

| KPI | 설명 |
|------|------|
| 발급률 | 100% (전체 발급) |
| 사용률 | 사용/발급 (색상 코딩) |
| 총 발급 수 | 건수 |
| 총 사용 수 | 건수 |

#### 쿠폰 사용 랭킹 TOP 5

| 항목 | 설명 |
|------|------|
| 순위 | 1~5위 뱃지 (금/은/동/기본) |
| 쿠폰명 | 이름 |
| 발급 수 | 건수 |
| 사용 수 | 건수 |
| 사용률 | 퍼센트 + 프로그레스 바 |
| vs 평균 | 평균 대비 차이 |

#### 성과 인사이트

- 초록 성공 박스: 잘되고 있는 항목
- 추천 액션: 유지/확장 제안
