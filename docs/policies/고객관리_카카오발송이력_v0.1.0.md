## A. 고객 리스트(어드민) - 고객화면

### ▼ 범위 요약

#### 포함 범위

| 영역 | 기능 | 비고 |
|------|------|------|
| 어드민 | 고객 리스트 | 검색/필터/정렬 |
| 어드민 | 고객 상세 패널 (슬라이드) | 우측 슬라이드 패널 |

| 어드민 | 접속 정보 | 브라우저ID/기기/유입경로/최종접속/접속횟수 |
| 어드민 | **카카오 발송 이력** | NEW 뱃지, 발송 내역 테이블 |

---

### ▼ A-1. 고객 리스트

- 메뉴: 고객 관리 > 고객 리스트
- 테이블 항목: NO, 고객ID, 이름, 이메일, 마케팅 수신 동의, 최종 접속일, 접속 횟수
- 마케팅 수신 동의: 동의(초록 뱃지) / 미동의(기본 뱃지)
- 행 클릭 시 우측 상세 패널 오픈

---

### ▼ A-2. 고객 상세 패널

- 위치: 우측 슬라이드 패널 (overlay)
- 닫기(X) 버튼

#### 기본 정보

| 항목 | 표시 |
|------|------|
| 이름 | 고객명 |
| 성별 | 남성/여성 |
| 생년월일 | YYYY.MM.DD |
| 연락처 | 010-XXXX-XXXX |
| 주소 | 주소 텍스트 |
| 이메일 | email@example.com |

#### 접속 정보

| 항목 | 표시 | 비고 |
|------|------|------|
| 브라우저 ID | 복사 가능 (클립보드) | 복사 아이콘 |
| 고객 ID | ID 텍스트 | - |
| 접속 기기 | 데스크톱/모바일 | 아이콘 |
| 유입 경로 | URL | - |
| 최종 접속 | YYYY.MM.DD HH:MM | - |
| 접속 횟수 | 회 | - |
| 마케팅 수신 동의 | 동의/미동의 | 뱃지 |

---

### ▼ A-3. 카카오 발송 이력 (NEW)

- 설명: 해당 고객에게 발송된 카카오 메시지 이력
- **NEW** 뱃지 표시 (제목 옆)

#### 테이블 항목

| 항목 | 설명 | 비고 |
|------|------|------|
| 일시 | 발송 일시 (YYYY.MM.DD HH:MM) | - |
| 유형 | 캠페인 유형 | kakao 뱃지 (앰버 배경) |
| 회차 | 1회 또는 반복(N/M회차) | 1회=기본 뱃지, 반복=info 뱃지 |
| 발송건수 | 발송수/총수 (퍼센트) | 예: 24/100 (24%) |
| 상태 | 발송 상태 | 뱃지 |

#### 상태별 뱃지

| 상태 | 뱃지 variant | 색상 |
|------|------------|------|
| 진행 | success | 초록 |
| 실패 | destructive | 빨강 |
| 종료 | default | 회색 |
| 예정 | info | 파랑 |
| 중지 | warning | 노랑 |

#### 실패 상태 툴팁

- 실패 뱃지 호버 시 실패 사유 툴팁 표시
- 예: "포인트 잔액 부족", "수신 거부 고객"

#### 페이지네이션

- 5건/페이지
- 이전/다음 버튼 + 페이지 번호
- "총 N건" 표시
