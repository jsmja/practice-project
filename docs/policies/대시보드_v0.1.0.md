### ▼ A-1. 긴급 알림 배너

- 설명: 서비스 장애, 데이터 경고 등 긴급 상황을 대시보드 최상단에 표시
- 위치: 대시보드 최상단 (모든 콘텐츠 위)
- 닫기(X) 가능 (세션 내 dismiss)
- severity 2 이상만 상단 배너에 표시
- 최대 표시 개수: 5개 (초과 시 우선순위 순 상위 5개)
- 동일 대상 중복 방지: 여러 임계값 해당 시 상위 타입만 표시

#### 심각도 등급

| severity | 라벨 | 색상 | 아이콘 | 배너 표시 |
|:---:|------|------|--------|:---:|
| 3 | **서비스 중단 위험** | 빨강 (red-50, border-l-red-500) | AlertCircle | O |
| 2 | **데이터 경고** | 노랑 (amber-50, border-l-amber-500) | AlertTriangle | O |
| 1 | 개선 제안 | - | - | X (운영 알림 섹션에서만 표시) |

#### 케이스 정의: 서비스 중단 위험 (severity 3, destructive)

| 카테고리 | 조건 | 메시지 예시 | 액션 |
|---------|------|----------|------|
| 서비스 연동 | 연동 토큰 만료 | "{서비스명} 연동이 만료되었습니다. 재연결이 필요합니다." | 서비스 연결 |
| 서비스 연동 | API 응답 오류 연속 3회 이상 | "{서비스명} 서비스에 오류가 발생했습니다. 상태를 확인해주세요." | 서비스 연결 |
| 포인트 | 잔여 포인트 ≤ 1,000P | "잔여 포인트가 {N}P입니다. 발송이 중단될 수 있습니다." | 충전하기 |
| 수신 동의 | 수신 동의 고객 ≤ 10명 | "마케팅 수신 동의 고객이 {N}명뿐입니다. 캠페인 발송이 제한됩니다." | 수신 동의 관리 |
| 캠페인 성과 | 클릭률/전환율 전주 대비 ≥ 10% 하락 | "{캠페인명} 클릭률이 전주 대비 {N}% 하락했습니다." | 캠페인 분석 |
| 캠페인 성과 | 발송 실패율 ≥ 5% | "발송 실패율이 {N}%를 초과했습니다. 수신자 목록을 확인하세요." | 발송 현황 |

#### 케이스 정의: 서비스 중단 위험 (severity 3, warning)

| 카테고리 | 조건 | 메시지 예시 | 액션 |
|---------|------|----------|------|
| 포인트 | 잔여 포인트 ≤ 5,000P | "잔여 포인트가 {N}P 이하입니다. 충전이 필요합니다." | 충전하기 |

#### 케이스 정의: 데이터 경고 (severity 2, destructive)

| 카테고리 | 조건 | 메시지 예시 | 액션 |
|---------|------|----------|------|
| 캠페인 성과 | 클릭률/전환율 급락 (전주 대비 ≥ 10%) | "{캠페인명} 클릭률이 전주 대비 {N}% 하락했습니다." | 캠페인 분석 |

#### 케이스 정의: 데이터 경고 (severity 2, warning)

| 카테고리 | 조건 | 메시지 예시 | 액션 |
|---------|------|----------|------|
| 서비스 연동 | 연동 토큰 만료 임박 (D-7 이내) | "{서비스명} 연동이 {N}일 후 만료됩니다. 갱신이 필요합니다." | 서비스 연결 |
| 수신 동의 | 수신 동의 고객 ≤ 50명 | "마케팅 수신 동의 고객이 {N}명 이하입니다. 수신 동의 유도 캠페인을 설정하세요." | 수신 동의 관리 |
| 수신 동의 | 수신 동의 이탈률 ≥ 5% (주간) | "이번 주 수신 동의 해지율이 {N}%입니다. 원인을 확인해주세요." | 수신 동의 관리 |
| 캠페인 운영 | 캠페인 종료 임박 (D-3 이내) | "{캠페인명} 캠페인이 {N}일 후 종료됩니다." | 캠페인 관리 |

#### 케이스 정의: 개선 제안 (severity 1, info) — 배너 미표시, 운영 알림에서만

| 카테고리 | 조건 | 메시지 예시 | 액션 |
|---------|------|----------|------|
| 서비스 연동 | 서비스 미연동 상태 | "{서비스명}을 연동하면 마케팅 채널을 확장할 수 있습니다." | 서비스 연결 |
| 캠페인 운영 | 활성 캠페인 0개 | "진행 중인 캠페인이 없습니다. 새 캠페인을 시작하세요." | 캠페인 만들기 |
| 캠페인 성과 | 성과 정체 (7일간 변동 없음) | "{캠페인명} 성과가 7일간 변동이 없습니다. 메시지 교체를 고려해보세요." | 캠페인 관리 |
| 시스템 | 신규 기능 출시 | "{기능명} 기능이 추가되었습니다." | 기능 보기 |

#### 메시지 작성 규칙

| 요소 | 규칙 | 예시 |
|------|------|------|
| 치환 변수 | `{변수명}` → 런타임 치환 | `{서비스명}`, `{캠페인명}`, `{N}` |
| 문장 구조 | 현상 서술 + 행동 유도 (2문장 이내) | "~ 만료되었습니다. 재연결이 필요합니다." |
| 수치 포함 | 임계값 알림은 반드시 현재 수치 포함 | "잔여 포인트가 **820P**입니다." |
| 톤 | destructive: 긴급 / warning: 권유 / info: 안내 | "~ 중단될 수 있습니다" / "~ 필요합니다" |
