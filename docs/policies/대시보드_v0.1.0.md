### ▼ A-1. 긴급 알림 배너

- 설명: 서비스 장애, 데이터 이상 등 긴급 상황 안내
- 위치: 대시보드 최상단
- 유형별 색상:
  - 서비스 장애: 빨간 배경
  - 데이터 이상: 노란 배경
- 닫기(X) 가능
- 정책 기준: `ALERTS_AND_RECOMMENDATIONS_POLICY.md` 참조

---

### ▼ A-2. 서비스 요약

- 설명: 핵심 서비스 현황을 카드 형태로 요약

| 카드 | 표시 항목 |
|------|---------|
| 헤이보드 | 활성 카드 수, 오늘 클릭, 상태 뱃지 |
| 채팅상담 | 오늘 상담 건수, 대기 건수, 잔여 쿼터 |
| 마케팅 | 진행 중 캠페인, 이번 달 발송, 평균 오픈률 |
| 포인트 잔액 | 현재 잔액, 잔액 상태 (정상/주의/부족) |

---

### ▼ A-3. 채널 연동 상태 바

- 설명: 연동 서비스의 활성/비활성 상태를 한줄로 표시

| 서비스 | 표시 |
|--------|------|
| 카페24 | ✓ 연동됨 (초록) |
| 인스타그램 | ✗ 미연동 (빨강) |
| 유튜브 | ✓ 연동됨 |
| 채팅상담 | ✓ 연동됨 |
| 헤이보드 | ✓ 활성 |
| 마수동 | ✓ 활성 |

- 클릭 시 서비스 연동 페이지로 이동

---

### ▼ A-4. 헤이보드 성과

- 클릭수 / 평균 체류시간
- 인기 카드 TOP 3 (카드명, 클릭수, 전환율)

---

### ▼ A-5. 배너 통계 + 마케팅 고객 (2컬럼)

#### 배너 통계
- 운영 중 배너 수, 총 노출, 총 클릭, 평균 CTR

#### 마케팅 고객
- 총 고객 수, 마케팅 수신 동의 수, 동의율

---

### ▼ A-6. CRM 퍼널 시각화

- 설명: 메시지 발송 성과를 퍼널 형태로 요약

| 단계 | 항목 |
|------|------|
| 발송 | 총 발송 건수, 발송률 |
| 열람 | 열람 건수, 열람률 |
| 클릭 | 클릭 건수, 클릭률 |
| 전환 | 전환 건수, 전환률 |

- 각 단계별 업종 평균 범위 표시
- 평균 대비 평가 뱃지 (우수/양호/개선필요)

---

### ▼ A-7. 진행 중 캠페인

- 최근 진행 중인 캠페인 리스트
- 항목: 캠페인명, 유형 뱃지, 발송일, 상태, 발송/대상 수
- 복사/일시중지 액션 버튼

---

### ▼ A-8. AI 추천

- 설명: AI 기반 마케팅 개선 추천
- 추천 카드 형태 (아이콘 + 제목 + 설명)

---

### ▼ A-9. 사이트 개요

- 오늘 방문자 / 페이지뷰 / 평균 체류시간 / 이탈률
- 기기별 비율 (모바일/데스크톱/태블릿)
- 인기 페이지 TOP 3

---

### ▼ A-10. 운영 알림

- 결제/시스템/보안 관련 알림 목록
- 알림 유형별 아이콘 + 시간 표시
