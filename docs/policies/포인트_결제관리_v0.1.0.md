### ▼ A-1. 포인트 관리 페이지

- 메뉴: 결제/포인트 > 포인트 관리

#### 화면 구성

##### 잔액 경고 배너
- 잔액 ≤ 50,000P: 노란 배경, "잔액이 50,000P 이하입니다"
- 잔액 ≤ 10,000P: 빨간 배경, "잔액이 부족합니다. 충전 후 발송 가능"

##### 포인트 현황 카드 (좌측 2열)

**환불 대기 없을 때:**
```
잔액        48,500P
[충전] [환불]
```

**환불 대기 있을 때:**
```
사용 가능      18,500P
총 48,500P 중 환불 대기 30,000P 보류
▼ 환불 대기 1건 · 30,000P (접기)
  ├ 상세 내역 + [취소] 버튼
  └ 영업일 3~5일 이내 처리
[충전] [환불 처리중 (비활성)]
```

##### 발송 단가 카드 (우측 2열)
- 25P/건 (= 25원/건, 기본형·와이드 이미지형 동일)
- 안내: 실패 시 자동 환불 / 잔액 부족 시 발송 중단 / 유효기간 1년

##### 포인트 내역 테이블
- 기간 필터: 전체/주/월
- 타입 필터: 전체/충전/차감/환불/환불신청/만료
- 항목: NO, 일시, 구분(뱃지), 상세, 변동, 잔액

---

### ▼ A-2. 포인트 충전

#### 충전 모달

##### 1단계: 금액 선택
- 프리셋: 1만 / 3만 / 5만 / 10만 / 30만원 + 직접 입력
- 직접 입력: 10,000원 단위, 최소 10,000원
- 결제 수단 선택
  - 기존 등록 카드 (신한카드 ····1234) — 체크 아이콘
  - 신규 카드로 결제 — 점선 테두리
- 결제 금액 표시

| 항목 | 금액 |
|------|------|
| 충전 금액 | 50,000원 |
| 수수료 (10%) | 5,000원 |
| **총 결제 금액** | **55,000원** |
| 충전 포인트 | 50,000P |

##### 2단계: 결제 확인
- 충전 포인트, 충전 금액, 수수료, 총 결제 금액, 결제 수단, 유효기간

##### 3단계: 충전 완료
- "50,000P 충전 완료!" + 현재 잔액 표시

### ▼ A-3. 포인트 환불

#### 환불 모달

##### 환불 방식 선택

| 방식 | 처리 | 금액 | 뱃지 |
|------|------|------|------|
| 카드 즉시 환불 | 즉시 | 전액만 | 즉시 처리 (초록) |
| 계좌 환불 | 3~5일 | 부분 가능 | 3~5일 소요 (노랑) |

##### 카드 즉시 환불
- 환불 포인트 입력 (10,000P 단위)
- 환불 사유 선택 (서비스 미사용/구독 취소/이중 결제/기타)
- → "즉시 환불 처리되었습니다"

##### 계좌 환불
- 환불 포인트 + 사유 + 은행 선택 + 계좌번호 + 예금주
- → "환불 신청이 접수되었습니다. 운영팀 검토 후 3~5일 이내 처리"

##### 환불 제한
- 복수 환불 불가: 처리 중 건이 있으면 버튼 비활성화 ("환불 처리중")
- 환불 대기 금액 = 사용 가능 잔액에서 차감
- 환불 신청 취소 가능 (접기 상세에서)

## B. 결제내역(어드민) - 고객화면

### ▼ B-1. 결제내역 페이지

- 메뉴: 결제/포인트 > 결제내역

#### 검색 기간
- 날짜 범위: 시작일 ~ 종료일 (date picker)
- 퀵 필터: 전체 / 주 / 월
- 조회 N건 표시

#### 합계 카드 (3종)

| 카드 | 내용 |
|------|------|
| 결제/충전 합계 | +889,990원 (검정) |
| 환불/취소 합계 | -60,000원 (빨강) |
| 순 결제 금액 | +829,990원 (보라) |

#### 상태별 탭

전체 | 결제완료 | 충전완료 | 환불대기 | 환불완료 | 결제취소 | 결제실패

#### 리스트 항목

| 항목 | 설명 |
|------|------|
| 결제 일시 | 결제 처리 일시 |
| 상태 | 뱃지 (결제완료=초록, 충전완료=파랑, 환불대기=노랑, 결제실패=빨강 등) |
| 구독 상품 | 서비스명 또는 포인트 충전/환불 |
| 구독 유형 | 월간/연간 구독 또는 - |
| 결제 수단 | 카드 정보 또는 계좌 정보 |
| 결제 금액 | 금액 (환불은 빨간색 음수) |
| 사유 | 수수료 포함, 환불 요청, 한도 초과 등 |
| 영수증 | [보기] 링크 |
| 처리 | 환불대기 건만 [환불 처리] 버튼 |

#### 상태별 정의

| 상태 | 설명 |
|------|------|
| 결제완료 | 구독 정상 결제 |
| 충전완료 | 포인트 충전 (수수료 포함 금액) |
| 환불대기 | 계좌 환불 신청 (운영팀 승인 대기) |
| 환불완료 | 환불 처리 완료 |
| 결제취소 | 구독 해지 전액 환불 |
| 결제실패 | 결제 실패 (한도 초과 등) |
